# Smart Project Manager 

## Project Overview
An enterprise-grade project management application that integrates JavaFX (GUI), Oracle Database 21c (Dockerized), and Predictive AI analytics, focusing on high operational resilience against technical infrastructure failures.

## The 3 Core Pillars
### AI-Driven Delay Prediction:
Dynamically scans database tasks, compares estimated vs. actual execution hours, and calculates a precise percentage tracking the continuous risk of project timeline delays.
## Cyber Resilience & Fault Tolerance:
An automated fallback mechanism; if the core Oracle server drops or encounters latency, the application autonomously diverts and logs runtime entries into a local file to ensure zero data loss and uninterrupted operations.
## Containerized Database Infrastructure:
Leverages an isolated Oracle 21c schema deployed inside a Docker Container, utilizing Pluggable Database (XEPDB1) architecture for professional, scalable data isolation.
